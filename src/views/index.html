Poker Planner app for Slack


Star Issue

 
 

What is this?

This is a free and open-source project lets you make estimating and planning just from Slack, without any need of external software or web-app. You can read more planning poker on wiki. This service is totally free, you can install the app with "Add to Slack" button above. If you want to host your own app, please check the setup guide on Github.

Usage

After successful installation, /pp command will be avaliable in your slack team. Just type /pp some task name with a short description to start a poker planning. This command will find active users in the current channel/group and add them as participants to poker planning. (Therefore /pp command does not work in direct messages.)

Alternatively you can specifically add users to poker planning just by mentioning them: /pp some task name @user1 @user2
